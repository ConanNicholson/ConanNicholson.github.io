CentOS安装过程详解

写一下图形安装CentOS的步骤，方便自己记忆，也希望能帮助别人。

步骤从进入到图形安装界面开始，怎么用镜像文件安装CentOS前面文章已经说过，在这里不再赘述。

开始安装

Install or upgrade an existing system 安装或升级现有的系统

install system with basic video driver 安装过程中采用基本的显卡驱动

Rescue installed system 进入系统修复模式

Boot from local drive 退出安装从硬盘启动

Memory test 内存检测

选择的话，先选择第一项，如果第一项不能执行安装的话，就试试第二项。

这里可以选择跳过。

可以选择中文，也可以选择英文，英文兼容性更好点，这个是一个比较有挑战性的选项对于英语不好的人来说。

键盘布局选择英文。

如果是将CentOS 6安装到本地硬盘上，选择 Basic Storage Devices，如果安装到网络存储介质如SANs上，选择 Specialized Storage Devices

一般情况下选择第一项.

如果是新硬盘或硬盘上的数据已不再需要，选择 yes,discard any data，否则选择 No, keep any data

一般装系统，系统盘应该不会有重要数据，所以建议选择yes,discard any data。

Hostname填写格式“英文名.姓”，或者保持默认。

不要下一步，这里还有网络需要配置。

如果不配置的话，默认是不自动连接网络的，需要手动配置。

时区，国内选上海，国外因自己地域选择。

设置root的密码，如果提示密码弱，你可以选择无论如何都用，你也可以改复杂。学习的话，这个无所谓。真实情况下，建议根密码复杂。

下面是硬盘分区。

这里有两种方式分区。

  • 自动，选择第一项。
  • 手动，选择最后一项。

这个选择根据个人情况而定。

自动分区：

调整分区，必须要有/home这个分区，如果没有这个分区，安装部分软件会出现不能安装的问题。

自动分区的话，都会分好/home分区。

下一步，格式化分区。

然后，将分区更改写入硬盘。

这里需要注意一下，之前我安装的时候，用U盘安装，引导位置选择错了，导致将系统引导安装到U盘上了。

所以，到这步的时候，根据自己情况，如果只有一个硬盘那就默认就可以。如果有别的硬盘或者U盘就需要注意一下这里，选择系统安装的硬盘。

手动分区：

手动分区方案可以参考这篇文章：Linux分区方案

这一步，有两种方案：

  1. 根据自己的需要选择安装包。
  2. 最小化安装。

第一种的话，如果当桌面系统，就选第一项或者第二项。

第二种，选择“Minimal”并勾选复选框“Customize now”配置需要安装的软体。安装Linux时没有必要把所有的软体都选上，只需要安装需要的Packages就可以了。其它软体则在后续需要时再安装。系统的超强可配置性，是Linux的优点之一。

这里只选择，“Base System”=>“Base”(系统基础安装包) 以及 “Desktop的三个安装包”。

然后就是等待系统安装完成。

系统安装完成后面的设置很简单，这里就不说了。
