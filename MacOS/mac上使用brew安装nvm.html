Mac上使用brew安装nvm

brew方式

如果机器没有安装过node，那么首先安装nvm。

brew install nvm

其次需要在shell的配置文件(~/.bash_profile)中添加如下内容：

# For NVM
export NVM_DIR=~/.nvm
source $(brew --prefix nvm)/nvm.sh
