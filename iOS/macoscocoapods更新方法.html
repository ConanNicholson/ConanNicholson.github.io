MacOSCocoapods更新方法

执行

sudo gem install cocoapods --pre

升级Cocoapods，报如下错误。

sudo gem install cocoapods --pre
ERROR:  While executing gem ... (Errno::EPERM)
    Operation not permitted - /usr/bin/pod

改为输入如下命令，则能成功升级。

sudo gem install -n /usr/local/bin cocoapods --pre
