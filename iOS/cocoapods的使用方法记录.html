cocoapods的使用方法记录

查看源

gem source

删除源

gem sources --remove 源地址

添加源

gem source -a https://gems.ruby-china.org/

安装cocoapods

sudo gem install -n /usr/local/bin cocoapods --pre

此步骤常见问题：

1.

ERROR:  While executing gem ... (Gem::DependencyError)
    Unable to resolve dependencies: cocoapods requires cocoapods-core (= 0.33.1), claide (~> 0.6.1), cocoapods-downloader (~> 0.6.1), cocoapods-plugins (~> 0.2.0), cocoapods-try (~> 0.3.0), cocoapods-trunk (~> 0.1.1), nap (~> 0.7)

解决方法：

https://rubygems.org/

上面的源更新一下gem

sudo gem update --system

2.

ERROR:  While executing gem ... (Errno::EPERM)
    Operation not permitted - /usr/bin/xcodeproj

解决方法：

sudo gem install -n /usr/local/bin cocoapods --pre

初始化cocoapods

pod setup

到项目目录初始化Podfile文件

pod init

不更新本地库信息进行安装

pod install --no-repo-updateorpod install

升级库

pod repo update

修改了Podfile文件，更新库

pod update --no-repo-updateorpod update

版本

pod --version

Podfile文件一般格式:

# Uncomment the next line to define a global platform for your project
# platform :ios, '9.0'

target 'xxxx' do
  # Uncomment the next line if you're using Swift or would like to use dynamic frameworks
  # use_frameworks!
    pod '库名'
  # Pods for xxxx

  target 'xxxxTests' do
    inherit! :search_paths
    # Pods for testing
  end

end
