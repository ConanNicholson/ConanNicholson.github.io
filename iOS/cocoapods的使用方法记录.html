cocoapods的使用方法记录

删除源

gem sources --remove 源地址

添加源

gem source -a https://gems.ruby-china.org

查看源

gem source

安装cocoapods

sudo gem install coocoapods

初始化cocoapods

pod setup

到项目目录初始化Podfile文件

pod init

不更新本地库信息进行安装

pod install --no-repo-updateorpod install

升级库

pod repo update

修改了Podfile文件，更新库

pod update --no-repo-updateorpod update

更新gem

sudo gem update

版本

pod --version

Podfile文件一般格式:

platform :ios, '8.0'
use_frameworks!
target 'DecorationBus' do
#此处添加库   
end
