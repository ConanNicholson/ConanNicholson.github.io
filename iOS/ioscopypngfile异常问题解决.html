IOSCopyPNGFile异常问题解决

在写代码的时候遇到Can't find xxx.png Command /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/usr/bin/copypng failed with exit code 1错误。

解决办法：

1.点击工程的TARGETS

2.点击Build Phases

3.点击Copy Bundle Resources

4.看里面的图片资源文件有什么是灰色的，然后选中它，最底下有删除的按钮或者del键。删掉再编译，应该就没问题了。

5.最后最好看看自己的图片文件是否有问题，图片不存在或者没有添加到编译器里，确认完成没问题再编译，运行。
